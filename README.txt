====================

What is this repo:

Replicates the Fair PCA paper, plus an application on yield curve data.

- ./data: contains 3 data sets (credit, images, and yield curves)

- main_*.py: run each to get plots and summary tables for each data set (credit, faces, and yield curves)

- *Process.py: data preprocess for each data set

- funcs.py: utils functions

====================

How to run:

- Run each main_*.py, they will create folders to store plots and tables in ./figs and ./tables
